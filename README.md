- 🏗️ I care deeply about **clean architecture** — code that stays maintainable years after it ships
- ⚡ I optimize for **performance**: query tuning, Redis caching, queue-driven workloads, and Elasticsearch at scale
- ☁️ I deploy and operate my own infrastructure on **AWS** (EC2, S3, RDS, CloudFront, Route53) with **Docker** and **CI/CD**

## 🧭 Development Philosophy

- 🧹 **Clean Code** — readable, intention-revealing code beats clever code
- 🧱 **SOLID Principles** — single responsibility and dependency inversion keep systems flexible
- ♻️ **Reusable Components** — build once, use everywhere, maintain in one place
- ⚡ **Performance First** — measure, profile, and optimize before scaling hardware
- 🔐 **Security First** — validate inputs, least privilege, secure defaults from day one
- 📐 **Scalable Architecture** — design for 10× the current load
- 📚 **Documentation** — a feature isn't done until the next developer can understand it
- 🧪 **Testing** — automated tests are the safety net that makes refactoring possible
- 🤖 **Automation** — if it's done twice manually, it should be scripted
